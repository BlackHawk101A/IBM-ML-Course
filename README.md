### IBM-Machine-Learning-Foundational-Course

This repository contains all files related to the personal projects made as part of the IBM Machine Learning Foundation Course series.

** Full reports present in ./docs/ **

## Projects

# Exploratory Data Analysis Project

A comprehensive analysis performed on the Titanic survival dataset

The aim was to formulate dependencies and uncover patterns in the dataset, as well as build meaningful relationships between feature and target variables, using various visualizations alongside hypothesis testing. These relationships represented the survivability of different demographics of people in the Titanic disaster.

# Supervised Machine Learning (Regression) Project

A detailed machine learning project to build various regression models on the Housing Prices dataset.

The aim was to build various different linear regression models to predict the prices of houses in the Boston area. Steps included preprocessing and cleaning the dataset as well as performing feature engineering, before testing various models and testing them against common metrics to compare performance.

# Supervised Machine Learning (Classification) Project

An insightful machine learning project building and testing various classification models on the Heart Disease dataset.

The aim was to build a complete end-to-end pipeline to predict whether a given patient has heart disease, from their medical information. The process included feature preprocessing and cleaning, after which a wide range of models and ensemble methods were built and scored using metrics such as F1 to find the optimum model and parameters.

# Unsupervised Machine Learning Project

An engaging project exploring various clustering methods to classify the flowers of the Iris dataset.

The aim was to build an test various clustering methods on the famous Iris dataset, based on information about the sepals and petals, also learning correlations between the features in the process. After preprocessing, multiple models based on different techniques were built, with results plotted and compared using the Silhouette score.
